# CLI Password Manager

# Key Features:
# 1. Add Password.
# 2. View Password.
# 3. Delete Password.
# 4. Change Password.
# 5. Exit
# 6. Stores Website/App Name.
# 7. Stores Username and Password.
# 8. Save data to a file.
# 9. Load data when program starts.
# 10. Limited to only one account per website.


# Future Improvements:
# 1.Making it capable of storing more than one account per website.
# 2. Using Encryption over the password manager.
